
# Image Classification

This project is a deep learning image classification task using TensorFlow and Keras.
It includes training a CNN on a custom image dataset.


## Run Locally

Clone the project

```bash
git clone https://github.com/ashuguptahere/parspec-assignment.git
```

Go to the project directory

```bash
cd parspec-assignment
```

Starting the Server, Frontend, and Installing Dependencies

```bash
python3 main.py
```


## Demo

You can start the frontend on ```http://localhost:5173/```
